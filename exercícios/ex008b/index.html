Código fonte

O comando document.getElementId("teste") está formatado com a tag CODE

Para que o código fique indentado use a tag PRE antes de CODE:


num = int(input('Digite um número'))
    if num%2==0:
        print(f'O número {num} é PAR')
    else
        print(f'O número {num} é IMPAR')
    print('fim do programa')

use shift + tab para levar o código à margem
